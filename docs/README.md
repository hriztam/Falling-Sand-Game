# Falling Sand Game Docs

This folder documents the structure and ideas behind the project.

Files:
- `docs/project-overview.md`: how the current codebase is organized
- `docs/cellular-automata.md`: the core simulation idea behind falling sand games
- `docs/material-sand.md`: how sand behavior works
- `docs/material-water.md`: how water behavior works
- `docs/sand-water-interaction.md`: how sand sinking through water works

Suggested reading order:
1. `docs/project-overview.md`
2. `docs/cellular-automata.md`
3. `docs/material-sand.md`
4. `docs/material-water.md`
5. `docs/sand-water-interaction.md`
